# 物理治療APP

目的：讓物理治療師在評估個案時，有可視覺化界面，用於和個案溝通，紀錄個案資訊
主要治療手法：徒手治療

### 需求

- 框架
    - 需要相容iOS, iPadOS, Android, Windows
    - 推薦我一個框架
- 介面瀏覽方式
    - 直式
        - 手機
        - 平板
    - 橫式
        - 手機
        - 平板
        - PC

## APP功能

### 人體模型

- 主要功能
    - 展示肌肉
    - 展示神經
    - 可活動，但`不可超出人體活動範圍`
    - 可轉視角
    - 顯示/隱藏各器官名
    - 能隱藏開關各器官部位
        - 分為表層肌群、深層肌群
- 2D版
    - 正面
    - 側面
    - 背面
- 3D版
    - 關節
        - 活動時會用顏色顯示肌群收縮、伸展
    - 單一標準版本（原精簡版/細節版雙版構想已收斂為單一資產，見 04 §4.3.5）
        - 逐肌肉明確線條、神經束
        - 完整無損版（full）為手動 opt-in；依裝置能力與 FPS 自動降級為安全網
